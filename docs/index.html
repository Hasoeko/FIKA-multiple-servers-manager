SPT / FIKA LAN Tool

FIKA Multiple Servers Manager

A lightweight local web controller for launching a single-player SPT server, a coop FIKA server, and one shared headless client from another PC on your network.

Quick Start

Run the web controller on the machine that hosts the SPT/FIKA installs, then open the configured port from another PC on the same network.

C:\Games\EscapeFromTarkov\Web Server\Start-SPT-Web-Control.cmd

Local Use Notes

This tool is intended for trusted LAN use. Change the default password in Web Server\config.json, keep the port private, and avoid exposing it directly to the internet.
